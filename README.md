# Hivobu

This is a simple tool based on an idea from the Bebras Challenge from 2025, namely the “Hivobu” task. It allows you to type in a code with a specific syntax based on a postfix notation for unary and binary operators, and then it generates an SVG image based on that code.

A live demo is available at <http://jp.pellet.name/hep/hivobu/>.

## Usage

Use the cheat sheet at the bottom of the page to see the syntax for the code.

Basically, you have a series of primitive shapes (square, circle, triangle) that you can combine using various operators (place next to, place on top of, place under, etc.). You can also apply transformations to the shapes (rotate, apply color, etc.). The code is written in a postfix notation, which means that the operators come after their operands.

## Features

The URL is updated with the code you have typed in, so you can easily share your creations by sharing the URL. Browser history items are created for each “compiling” code, so navigation between different versions is possible. <key>Cmd+Shift+C</key> copies the generated image to the clipboard.
